Users Manual
============

Target group: **Editors**

-----------------
How does it work?
-----------------

Copy the url of your soundcloud playlist
 .. image:: ../Images/UserManual/getSoundcloudLink.png
    :alt: Soundcloud URL should be like https://soundcloud.com/xyUser/sets/xyPlaylist
  Do not use the url which you get when using the emebed share->embed

Go to your Backend to the Filelist
  .. image:: ../Images/UserManual/goToUserFileList.png

Add a new file and paste the soundcloud url
  .. image:: ../Images/UserManual/pasteLinkToSourceURL.png
      :alt: Soundcloud URL should be like https://soundcloud.com/xyUser/sets/xyPlaylist
  Soundcloud URL should be like ``https://soundcloud.com/xyUser/sets/xyPlaylist``

After that it should look something like this
  .. image:: ../Images/UserManual/viewAfterAdded.png

Now you can add it to a Text & Media Element of your choice just like a Youtube Video
 .. image:: ../Images/UserManual/goToTextAndMediaElement.png
