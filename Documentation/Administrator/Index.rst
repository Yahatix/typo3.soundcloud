.. _admin-installation:

Installation
------------

- The easiest way to install the extension is with composer
- There are no dependencies except for the typo3 version but it should work in any future version of typo3 as well

To install the extension, perform the following steps:

#. Run ``composer require yahatix/soundcloud``
#. Go to Extension Manager
#. Enable the extension 'soundcloud'
